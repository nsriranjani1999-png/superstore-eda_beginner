# Superstore Sales Data Analysis (EDA)

## Objective
To analyze retail sales data and identify trends, profitable segments, and loss-making areas.

## Dataset
Superstore sales dataset (CSV)

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Identified top-performing categories and sub-categories
- Analyzed profit trends across regions
- Detected products with high sales but negative profit
